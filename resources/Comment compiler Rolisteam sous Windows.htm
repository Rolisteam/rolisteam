Comment compiler Rolistik sous Windows XP ou Vista?

Vous devez avant toute chose être en mesure de compiler une application Qt 4.2.3 avec Dev-C++. Pour ma part j'ai trouvé toutes les explications ici : http://www.siteduzero.com/tuto-3-10101-1-installer-qt-sous-windows.html

Vous trouverez Qt ici : http://trolltech.com/developer/downloads/qt/windows

Et Dev-C++ là : http://www.bloodshed.net/dev/devcpp.html

Une fois Dev-C++ et Qt 4.2.3 installés, vous devez inclure le fichier qt.mak (fourni avec les sources) dans le Makefile (Projet > Options du projet (alt+p) > onglet "Makefile").

Il est préférable de recompiler Qt en statique de manière à permettre un échange plus simple de l'exécutable (il n'est ainsi plus nécessaire de fournir les DLL de Qt avec l'exécutable).

Vous pouvez maintenant compiler Rolistik à l'aide du fichier projet Rolistik.dev.

PS : n'oubliez pas qu'à la fin de l'édition de liens le compilateur s'arrête sur des erreurs "vtable", vous devez alors rajouter dans le projet les fichiers moc_* qui auront été générés pendant la compilation, avant de recompilier.

PPS : le fichier de ressources rolistik.rcc est généré par la commande rcc -binary rolistik.qrc -o rolistik.rcc
