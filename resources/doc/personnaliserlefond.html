Personnaliser le fond
 Previous Top Next
Il est possible de remplacer l'image de fond de l'espace de travail par une image de votre choix. Cette image doit être au format BMP et se nommer Rolisteam.bmp.
  • Sous Windows : placez l'image Rolisteam.bmp dans le répertoire de l'application (par défaut C:\Program Files\Rolisteam).
  • Sous MacOS : placez l'image Rolisteam.bmp dans le répertoire .Rolisteam se trouvant dans le répertoire de l'utilisateur. Pour ouvrir le répertoire .Rolisteam, ouvrez le finder et placez-vous dans le dossier par défaut de l'utilisateur, cliquez sur "Aller au dossier..." dans le menu "Aller", et tapez .Rolisteam dans la boîte de dialogue
  • Sous GNU/Linux : placez l'image Rolisteam.bmp dans le répertoire .Rolisteam se trouvant dans le répertoire de l'utilisateur (/home/USER/.Rolisteam/).

Au lancement l'application vérifiera la présence du fichier Rolisteam.bmp : s'il est présent celui-ci sera utilisé comme fond pour l'espace de travail; dans le cas contraire le fond par défaut sera employé.

L'image est toujours calée en haut à gauche de l'espace de travail.
