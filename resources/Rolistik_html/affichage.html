-      Afficher noms PJ : si la ligne est cochée alors les noms des PJ s'affichent sous le disque qui les représente sur les plans. Dans le cas contraire le nom du PJ n'est pas indiqué.

-      Afficher noms PNJ : si cochée, les noms des PNJ sont affichés sous le disque qui les représente.

-      Afficher numéro de PNJ : affiche ou masque les numéros des PNJ, et ce pour l'ensemble des PNJ et sur tous les plans.
